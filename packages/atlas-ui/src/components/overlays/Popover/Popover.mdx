<DocsLayout>

<DocsTitle>Popover</DocsTitle>

<DocsIntro>
A lightweight, non-modal floating panel anchored to a trigger. Source:
Atlas Product's Analytics page header — the "May 12 – May 18, 2025"
date-range button and the "Filters" button are exactly this pattern.
Built on Radix's Popover primitive — no focus trap (unlike Modal/
Drawer), since it isn't a modal interruption; dismissed by an outside
click or Escape.
</DocsIntro>

<DocsSection title="Overview">

`Popover` (root), `PopoverTrigger`, `PopoverAnchor` (for anchoring to a
different element than the trigger), and `PopoverContent`.

</DocsSection>

<DocsSection title="Live Preview">

<Canvas of={PopoverStories.Default} />

</DocsSection>

<DocsSection title="Usage">

Use Popover for a small, self-contained control anchored to a button —
a date-range picker, a filter panel, a quick-edit form. For anything
that needs to trap focus and interrupt the page (a form the user must
resolve), use Modal or Drawer instead.

</DocsSection>

<DocsSection title="Variants">

<Canvas of={PopoverStories.FilterPanel} />

</DocsSection>

<DocsSection title="Sizes">

No fixed size — set a width via `className` on `PopoverContent` to fit
the content (both examples above use a fixed width, matching the
Analytics date-range/filter buttons).

</DocsSection>

<DocsSection title="States">

Open/closed is the only state, shown via the trigger interaction in the
Live Preview.

</DocsSection>

<DocsSection title="Accessibility">

- Dismissed by clicking outside, pressing Escape, or (optionally)
  re-clicking the trigger — all handled by Radix.
- Unlike Modal, does not trap focus — appropriate for a lightweight,
  non-blocking control.

</DocsSection>

<ArgTypes of={PopoverStories} />

</DocsSection>

<DocsSection title="Examples">

```tsx
import { Calendar } from 'lucide-react';
import { Button } from '@/components/actions/Button';
import { Popover, PopoverContent, PopoverTrigger } from '@/components/overlays/Popover';

<Popover>
  <PopoverTrigger asChild>
    <Button variant="ghost" size="small" icon={<Calendar size={15} />}>
      May 12 – May 18, 2025
    </Button>
  </PopoverTrigger>
  <PopoverContent className="w-64">{/* date range picker */}</PopoverContent>
</Popover>
```

</DocsSection>
