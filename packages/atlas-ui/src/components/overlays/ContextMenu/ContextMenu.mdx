<DocsTitle>Context Menu</DocsTitle>

<DocsIntro>
A right-click menu. Shares Dropdown Menu's exact panel styling â€” same
`menuStyles` helpers â€” but triggered by a `contextmenu` event over an
area rather than a click on a visible trigger. Built on Radix's Context
Menu primitive.
</DocsIntro>

<DocsSection title="Overview">

`ContextMenu` (root), `ContextMenuTrigger` (the right-clickable area),
`ContextMenuContent`, and `ContextMenuItem`/`ContextMenuSeparator`/
`ContextMenuLabel`.

</DocsSection>

<DocsSection title="Live Preview">

<Canvas of={ContextMenuStories.Default} />

</DocsSection>

<DocsSection title="Usage">

Wrap the area that should respond to a right-click in
`ContextMenuTrigger` â€” a table row, a file card, a canvas region â€” and
list `ContextMenuItem`s inside `ContextMenuContent`, exactly like
Dropdown Menu.

</DocsSection>

<DocsSection title="Variants">

Context Menu shares every item variant (icon, shortcut, destructive,
disabled, label) with Dropdown Menu â€” see its documentation for
examples.

</DocsSection>

<DocsSection title="Sizes">

One size â€” the menu's width is determined by its content.

</DocsSection>

<DocsSection title="States">

Hover/keyboard-highlighted, disabled, and destructive item states match
Dropdown Menu exactly, since they share the same styling helpers.

</DocsSection>

<DocsSection title="Accessibility">

- Radix wires full `menu`/`menuitem` ARIA roles and keyboard navigation,
  the same as Dropdown Menu.
- Also opens via the keyboard context-menu key (Shift+F10 on Windows),
  not just an actual right-click, so keyboard-only users aren't excluded.

</DocsSection>

<DocsSection title="Examples">

```tsx
import { Pencil, Trash2 } from 'lucide-react';
import {
  ContextMenu,
  ContextMenuContent,
  ContextMenuItem,
  ContextMenuTrigger,
} from '@/components/overlays/ContextMenu';

<ContextMenu>
  <ContextMenuTrigger className="rounded-md border border-default p-4">
    Right-click this card
  </ContextMenuTrigger>
  <ContextMenuContent>
    <ContextMenuItem icon={<Pencil size={14} />}>Rename</ContextMenuItem>
    <ContextMenuItem icon={<Trash2 size={14} />} destructive>Delete</ContextMenuItem>
  </ContextMenuContent>
</ContextMenu>
```
