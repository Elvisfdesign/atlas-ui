# Sidebar

The app shell's primary left-hand navigation. Source: Atlas Product — the
Review Queue's sidebar (240px fixed width, icon + label nav rows, a
divider between the primary and secondary nav group, a profile card
pinned to the bottom).

## Overview

`Sidebar` is a structural shell — it owns the 240px width, the right-hand
divider, and vertical layout. `SidebarSection` groups a set of
`SidebarItem`s and can draw a divider above itself. The logo and the
bottom profile card are page-specific content, so they're composed as
plain children rather than baked-in props.

## Usage

Compose a `Sidebar` from one or more `SidebarSection`s, each holding
`SidebarItem`s. Use `withDivider` on a section that should visually
separate from the one above it (Product uses this once, between the main
nav group and Teams/Settings).

A `SidebarItem` is an icon, a label, and an optional trailing count chip.

## Variants

`SidebarItem` renders as a `<button>` by default, or a real `<a href>`
when you pass `href` — use the link form whenever the item is real
in-app navigation, so it behaves like a link (open in new tab, etc.).

<Canvas of={SidebarItemStories.ItemAsLink} />

## Sizes

One size — Sidebar's width (240px) and item height (35px) are fixed,
matching Atlas Product exactly.

## States

`active` marks the current section: `interactive-selected` background,
accent text — the same soft-indigo token the Data Table uses for a
selected row, so "this is the current one" reads consistently everywhere
in Atlas.

## Accessibility

- `Sidebar` renders as `<nav aria-label="Primary">`, distinguishing it
  from the page's other navigation landmarks (Breadcrumb, Tabs).
- The active item gets `aria-current="page"`.
- Both the button and link forms are fully keyboard operable natively.

## Examples

```tsx
import { Sidebar, SidebarSection, SidebarItem } from '@/components/navigation/Sidebar';
import { Home, Inbox, Settings } from 'lucide-react';

<Sidebar>
  <SidebarSection>
    <SidebarItem icon={<Home size={16} />} href="/">Home</SidebarItem>
    <SidebarItem icon={<Inbox size={16} />} href="/review-queue" active count={12}>
      Review Queue
    </SidebarItem>
  </SidebarSection>
  <SidebarSection withDivider>
    <SidebarItem icon={<Settings size={16} />} href="/settings">Settings</SidebarItem>
  </SidebarSection>
</Sidebar>
```
