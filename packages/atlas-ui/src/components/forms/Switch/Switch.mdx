# Switch

An on/off setting that takes effect immediately.

## Overview

Not yet its own componentized node in the Atlas UI System (see the React
Readiness matrix) — built from Atlas's motion and accent tokens (a
`duration-fast` slide, the same accent Checkbox and Radio use when
checked) as the immediate-effect sibling of Checkbox.

## Live Preview

<Canvas of={SwitchStories.Default} />

## Usage

Use Switch when toggling has an immediate, visible effect (auto-approve
is on right now) — not as part of a form that's submitted later, where
Checkbox is the right control.

## Variants

One visual treatment — Switch doesn't have a `variant` prop.

## Sizes

One size (20×36px).

## Accessibility

- Rendered with `role="switch"` on a real `<input type="checkbox">`, so
  screen readers announce it as a toggle with an on/off state, not a
  generic checkbox.
- The label is a real `<label htmlFor>`.
- Fully keyboard operable: Tab to focus, Space to toggle.
- The thumb's slide respects `prefers-reduced-motion` globally, same as
  every other Atlas transition.

## Examples

```tsx
import { Switch } from '@/components/forms/Switch';

<Switch
  label="Auto-approve high-confidence documents"
  checked={autoApprove}
  onChange={(e) => setAutoApprove(e.target.checked)}
/>
```
