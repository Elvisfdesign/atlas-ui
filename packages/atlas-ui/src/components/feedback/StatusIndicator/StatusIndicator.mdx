# Status Indicator

A dot + label pairing for a live/connection status. Distinct from Badge,
which is for categorical or process-state pills (Completed, Draft,
Review) rather than a live online/offline-style signal.

## Overview

A small colored dot beside a text label — status is always stated in
words, never left to the dot's color alone.

## Live Preview

<Canvas of={StatusIndicatorStories.Online} />

## Usage

Use for a person's presence (online/away/offline) or a connection's live
health (connected/disconnected) — anywhere the underlying idea is "is
this thing currently active." For a document or workflow's process
state, use Badge instead.

## Sizes

<Canvas of={StatusIndicatorStories.List} />

`sm`/`md` scale both the dot and the label text together.

## States

Status Indicator is a static display component — it has no interactive
states of its own.

## Accessibility

- The dot is `aria-hidden="true"` — the adjacent label text is the sole
  source of the status's meaning, so it reads correctly to screen reader
  users regardless of color perception.
