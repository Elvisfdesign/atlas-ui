<DocsLayout>

<DocsTitle>Skeleton</DocsTitle>

<DocsIntro>
The shimmer placeholder primitive. Source: the pulse treatment already
used ad hoc inside Table's `TableLoadingRows` and the chart components'
`loading` states — extracted here as its own reusable building block, so
any future loading placeholder composes it instead of writing another
`animate-pulse` div by hand.
</DocsIntro>

<DocsSection title="Overview">

A pulsing block shaped like the content it stands in for — text, a
circle (for an avatar), or a rectangle (for anything else).

</DocsSection>

<DocsSection title="Live Preview">

<Canvas of={SkeletonStories.Rect} />

</DocsSection>

<DocsSection title="Usage">

Compose several Skeletons together to mimic the shape of the real
content that's loading (a name + meta line, an avatar + two text lines).
Prefer Skeleton over Loading State when the destination layout is
already known — it reduces layout shift once real content arrives.

</DocsSection>

<DocsSection title="Variants">

<Canvas of={SkeletonStories.Text} />
<Canvas of={SkeletonStories.Circle} />

</DocsSection>

<DocsSection title="Sizes">

`width`/`height` are set directly — Skeleton has no fixed size scale,
since it needs to match whatever it's standing in for.

</DocsSection>

<DocsSection title="States">

<Canvas of={SkeletonStories.Composition} />

</DocsSection>

<DocsSection title="Accessibility">

- Always rendered `aria-hidden="true"` — a Skeleton has no content of
  its own to announce; pair it with a `role="status"`/`aria-live`
  container (or Loading State) at the level above it if you need loading
  to be announced.

</DocsSection>

<DocsSection title="React Props">

<Controls of={SkeletonStories.Rect} />

<ArgTypes of={SkeletonStories} />

</DocsSection>

<DocsSection title="Examples">

```tsx
import { Skeleton } from '@/components/feedback/Skeleton';

<div style={{ display: 'flex', gap: 12, alignItems: 'center' }}>
  <Skeleton variant="circle" width={36} height={36} />
  <div style={{ display: 'flex', flexDirection: 'column', gap: 8 }}>
    <Skeleton variant="text" width={140} />
    <Skeleton variant="text" width={100} />
  </div>
</div>
```
