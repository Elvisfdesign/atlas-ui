# Confidence Badge

Source: Atlas Product — Document Review's Extracted Fields list
(`field-row` → `fright`): a percentage beside a small tone-colored
circular icon chip.

## Overview

The compact form of a confidence readout — a percentage and a small
tone-colored icon chip, used beside a single value in a card or list row
where a full bar (Data Display's Confidence Indicator) would be visually
heavier than the context needs.

## Live Preview

<Canvas of={ConfidenceBadgeStories.Default} />

## Usage

Pass the raw 0–100 `value` — Confidence Badge rounds it and picks a tone
automatically using the same thresholds as Confidence Indicator (≥90
success, ≥70 warning, below danger).

## Variants

<Canvas of={ConfidenceBadgeStories.Warning} />
<Canvas of={ConfidenceBadgeStories.Danger} />

## Sizes

One size, sized to sit inline with a 13px value in Extracted Field Row.

## States

Static; Confidence Badge has no interactive states.

## Accessibility

- Confidence is conveyed by the percentage text, the icon, and the tone
  color together — never color alone (WCAG 1.4.1).
- The icon chip carries an `aria-label` with the full confidence
  sentence ("92% confidence") for assistive tech, since the chip alone
  is otherwise just a decorative icon.
