<DocsLayout>

<DocsTitle>AI Label</DocsTitle>

<DocsIntro>
Source: Atlas Product — the AI Assistant panel's "BETA" chip (Document
Review). A rectangular micro-tag for flagging an AI-powered feature or
its maturity, distinct from Badge's pill-shaped status tones.
</DocsIntro>

<DocsSection title="Overview">

A small uppercase tag, sized for sitting beside a heading or a piece of
AI-generated content.

</DocsSection>

<DocsSection title="Live Preview">

<Canvas of={AILabelStories.Default} />

</DocsSection>

<DocsSection title="Usage">

Use the default `neutral` tone to flag a feature as beta, new, or
experimental — matching the grounded BETA chip exactly. Use `accent` to
mark a specific piece of content as AI-generated, using the same
ai-bg/ai-border/ai-accent palette as AI Suggestion Card.

</DocsSection>

<DocsSection title="Variants">

<Canvas of={AILabelStories.Accent} />

</DocsSection>

<DocsSection title="Sizes">

One size — a compact tag meant to sit inline with text, not stand alone.

</DocsSection>

<DocsSection title="States">

Static; AI Label has no interactive states.

</DocsSection>

<DocsSection title="Accessibility">

- Purely visual. If the tag conveys information not otherwise available
  (e.g. "this value was AI-generated"), make sure surrounding text or an
  `aria-label` on the containing element conveys the same thing to
  assistive tech.

</DocsSection>

<Controls of={AILabelStories.Default} />

<ArgTypes of={AILabelStories} />

</DocsSection>

<DocsSection title="Examples">

```tsx
import { AILabel } from '@/components/ai/AILabel';

<h3>

</h3>

<p>
  Vendor Name <AILabel tone="accent">AI Generated</AILabel>
</p>
```
