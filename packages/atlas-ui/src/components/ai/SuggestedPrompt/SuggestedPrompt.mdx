# Suggested Prompt

Source: Atlas Product — the AI Assistant panel's `prompt-chip` rows
(Document Review): full-width, left-aligned suggestion buttons.

## Overview

A single clickable suggestion — clicking it sends its text as the
user's prompt. Assistant Panel renders a list of these above its
composer.

## Live Preview

<Canvas of={SuggestedPromptStories.Default} />

## Usage

Pass the suggestion text as `children` and handle `onClick` to send it
as a prompt — Assistant Panel does this automatically via its
`suggestedPrompts` prop.

## Variants

<Canvas of={SuggestedPromptStories.List} />

## Sizes

One size — full-width within its container.

## States

Default, hover, and keyboard focus (via `focus-visible`) — a real
`<button>`, so all native interaction states apply.

## Accessibility

- Renders as a real `<button>`, reachable and activatable by keyboard.
- The visible text is the accessible name — no separate label needed.

## Examples

```tsx
import { SuggestedPrompt } from '@/components/ai/SuggestedPrompt';

<SuggestedPrompt onClick={() => sendPrompt('Summarize this document')}>
  Summarize this document
</SuggestedPrompt>
```
