<DocsLayout>

<DocsTitle>Confidence Meter</DocsTitle>

<DocsIntro>
Not a direct 1:1 Figma frame — synthesized from a real recurring need:
Document Review's Approval Panel footer surfaces an aggregate figure
("96% avg. confidence") as plain text with no visualization. Confidence
Meter gives that aggregate a standalone, larger-scale visual home.
</DocsIntro>

<DocsSection title="Overview">

A large percentage number, a caption, and a wide Progress Indicator bar
— the same confidence → tone thresholds as Confidence Badge and Data
Display's Confidence Indicator, sized for a document- or workflow-level
summary rather than an inline table cell or list row.

</DocsSection>

<DocsSection title="Live Preview">

<Canvas of={ConfidenceMeterStories.Default} />

</DocsSection>

<DocsSection title="Usage">

Use Confidence Meter where a confidence figure deserves its own visual
weight — atop a document's field list, or in a workflow summary — rather
than sitting inline beside a single value (Confidence Badge) or a table
row (Confidence Indicator).

</DocsSection>

<DocsSection title="Variants">

<Canvas of={ConfidenceMeterStories.Warning} />
<Canvas of={ConfidenceMeterStories.Danger} />

</DocsSection>

<DocsSection title="Sizes">

One size — a full-width bar meant to anchor a card or section header.

</DocsSection>

<DocsSection title="States">

Static; Confidence Meter has no interactive states.

</DocsSection>

<DocsSection title="Accessibility">

- The bar carries `role="progressbar"` with `aria-valuenow`/`aria-valuemin`/`aria-valuemax`, composed from Data Display's Progress Indicator.
- Confidence is conveyed by the number, the label, and the tone color together — never color alone (WCAG 1.4.1).

</DocsSection>
