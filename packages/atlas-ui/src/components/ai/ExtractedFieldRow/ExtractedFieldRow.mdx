# Extracted Field Row

Source: Atlas Product — Document Review's Extracted Fields card
(`field-row`): a label + value stack on the left, a Confidence Badge on
the right.

## Overview

The single-row primitive Document Review's Extracted Fields card
repeats for every extracted value. Composes Confidence Badge rather
than reimplementing the percentage-and-chip treatment.

## Live Preview

<Canvas of={ExtractedFieldRowStories.Default} />

## Usage

Stack multiple rows in a card, separated by a divider, to build the
full Extracted Fields list.

## Variants

<Canvas of={ExtractedFieldRowStories.FieldList} />

## Sizes

One size, sized for a ~300px-wide fields sidebar.

## States

Static; row-level interactivity (editing a value) is left to the
consuming screen.

## Accessibility

- Label and value are plain text, readable by assistive tech in
  document order.
- Confidence Badge carries its own `aria-label` with the full
  confidence sentence.
