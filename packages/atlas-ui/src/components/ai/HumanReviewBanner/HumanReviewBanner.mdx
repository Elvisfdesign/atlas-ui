# Human Review Banner

Not a direct 1:1 Figma frame — synthesized from a real, recurring signal:
both Review Queue and Document Review mark documents with a "Review"
status whenever the AI's extraction needs a human to confirm it. Human
Review Banner gives that event a page-level, explanatory surface.

## Overview

An ai-tinted banner — icon, message, and an optional action — for
surfacing that the AI has flagged something specific for a human to
confirm, distinct from a generic warning banner.

## Live Preview

<Canvas of={HumanReviewBannerStories.Default} />

## Usage

Use Human Review Banner when the message is specifically attributed to
the AI's own assessment ("I'm not confident here"), not a generic system
warning — for those, use Feedback's Banner with a `warning` tone
instead.

## Variants

<Canvas of={HumanReviewBannerStories.WithAction} />

## Sizes

One size — a full-width banner meant to sit at the top of a document or
section.

## States

Static; Human Review Banner has no interactive states beyond its
optional action button.

## Accessibility

- `role="status"` announces the banner to assistive tech without
  interrupting, since it's informational rather than a page-blocking
  error.
- The icon is decorative (`aria-hidden`) — the message text alone
  carries the meaning.
