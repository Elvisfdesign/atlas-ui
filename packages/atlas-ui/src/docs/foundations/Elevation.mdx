# Elevation

Atlas uses shadow sparingly — most surfaces are flat, distinguished by a
hairline border and a subtle background-surface change, not a shadow. Only
things that visually float above the page get one.

## Where each is used

- **none** — cards, panels, table rows. The default for almost everything.
- **low** — dropdowns, menus.
- **medium** — popovers, tooltips.
- **high** — modals, dialogs.

There's a fifth concept, **overlay**, that isn't a shadow at all — it's the
dark scrim shown behind a modal or drawer (`overlay-scrim`, a semantic
color token, not part of this elevation scale).
