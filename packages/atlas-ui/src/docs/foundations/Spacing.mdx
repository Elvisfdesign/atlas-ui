# Spacing

A 4pt rhythm. Every gap and padding value in Atlas maps to this scale —
there is no such thing as a one-off `13px` margin in a component built
correctly.

<div style={{ display: 'flex', alignItems: 'flex-end', gap: '16px', marginTop: '24px', marginBottom: '32px' }}>
  {Object.entries(spacing).map(([key, value]) => (
    <div key={key} style={{ display: 'flex', flexDirection: 'column', alignItems: 'center', gap: '8px' }}>
      <div style={{ width: `${Math.max(parseInt(value, 10), 4)}px`, height: '32px', backgroundColor: 'var(--color-interactive-accent)', borderRadius: '2px' }} />
      <span style={{ fontFamily: 'monospace', fontSize: 11, color: 'var(--color-text-tertiary)' }}>{key}</span>
    </div>
  ))}
</div>

## Principle

Use an existing spacing variable whenever one fits. Only introduce a new
value when a genuine, repeatable gap has been found across multiple
places — never to make one component look slightly better in isolation.
