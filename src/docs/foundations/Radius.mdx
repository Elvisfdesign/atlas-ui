# Radius

Seven steps, from square corners to a full pill.

## Overview

Most of the system lives in three of them — `md` for interactive
elements, `lg` for containers, `full` for pills and circles. The rest
exist for the smaller number of cases that need something sharper or
rounder.

## Where each is used

- **none** — table cells, dense list rows.
- **xs (4px)** — small inline chips.
- **sm (6px)** — Small-size Button and Icon Button.
- **md (8px)** — Medium-size Button and Icon Button; the most common
  interactive-element radius in Atlas.
- **lg (12px)** — cards (KPI Card, panels), the unit of "this is a
  container."
- **xl (16px)** — large surfaces, modals.
- **full (999px)** — Badge, Avatar — anything that should read as a pill
  or a circle regardless of its content's width.
