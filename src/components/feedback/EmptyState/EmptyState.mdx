# Empty State

A centered icon-chip + title + description block for "nothing here yet."
Source: the same shape as Table's `TableEmptyState`, generalized into a
standalone, block-level component for use outside a table.

## Overview

An icon in a circular chip, a title, an optional description, and an
optional action — centered in its container. `tone` (`neutral` default,
`success`, `danger`) lets the same shape double as a Success or Error
state, rather than shipping three near-identical components for what is
visually one pattern.

## Usage

Use for an empty table, list, or search result (`tone="neutral"`, the
default). Use `tone="success"` for a completed/all-clear message, and
`tone="danger"` for a load/processing failure — override `icon` if the
default per-tone icon (Inbox/Check/AlertCircle) doesn't fit.

## Sizes

One size — Empty State fills its container's width and centers its
content within it.

## States

<Canvas of={EmptyStateStories.WithAction} />

`action` is optional — pass a Button when there's a clear next step
("Create workflow", "Try again"), omit it for a purely informational
empty state.

## Accessibility

- The icon chip is decorative (`aria-hidden="true"`) — the title and
  description carry all the meaning.
- Description text is capped at a readable width (`max-w-xs`) so it
  doesn't stretch into an unreadable single line in a wide container.
