# Checkbox

An on/off selection, usually as part of a group. Source: Atlas Product —
the Review Queue table's row-selection checkbox (17×17px, checked state
fills with the accent color and shows a white check).

## Overview

Built on a real native `<input type="checkbox">` (visually restyled)
rather than a custom button-based widget, so checked/indeterminate state,
labeling, and keyboard behavior all come from the browser.

## Usage

Use Checkbox for row selection in tables (its confirmed Product use), for
independent on/off options in a form, and for a "select all" control
paired with `indeterminate` when only some rows are selected. For a
setting that takes effect immediately rather than being part of a form
submission, use Switch instead.

## Variants

One visual treatment — Checkbox doesn't have a `variant` prop.

## Sizes

One size (17px), matching the Review Queue table checkbox exactly.

## Accessibility

- The label is a real `<label htmlFor>` — clicking anywhere on the label
  text toggles the checkbox, not just the box itself.
- `indeterminate` is set via the DOM property (not an HTML attribute,
  since none exists) so screen readers announce "partially checked"
  correctly.
- Fully keyboard operable: Tab to focus, Space to toggle, exactly like a
  native checkbox, because it is one.

## Examples

```tsx
import { Checkbox } from '@/components/forms/Checkbox';

<Checkbox
  label="Select all (3 of 12)"
  indeterminate={someSelected && !allSelected}
  checked={allSelected}
  onChange={(e) => toggleAll(e.target.checked)}
/>
```
